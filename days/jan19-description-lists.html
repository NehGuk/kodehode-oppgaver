Description list

Do not underestimate the basics! Although I had used ordered and unordered lists before, I was not familiar with description lists. They are defined with the <dl> tag. Description lists can have to types of child elements: the <dt> and the <dd>. They stand for, respectively, description term and description definition. A term can have several definitions, and more than one definition can be under the same term â€“ very much like a conventional dictionary, actually. For example: 


                  <dl>
                    <dt>Term 1</dt>
                    <dd>Description of term 1</dd>
                    <dt>Term 2</dt>
                    <dd>Description of term 2</dd>
                    <dt>Term 3</dt>
                    <dd>Description of term 3</dd>
                    <dd>Another description for term 3</dd>
                </dl>
